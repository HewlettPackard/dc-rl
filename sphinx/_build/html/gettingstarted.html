Getting Started

1. Basic example

Run the DCRL-Green environment to simulate one episode and view the roll-out information using the following command:

python dcrl_env.py

2. Train the agents

Run the following commands to simulate the DC enivronment and optimize the agents with default configurations:

2.1 Train using the DCRL environment

python train_ppo.py

2.2 Train using the DCRLeplus environment

If not done already, launch the docker container by executing the command:

docker run -t -i -v $PWD:/sinergym/dc-rl --shm-size=10.24gb agnprz/carbon_sustain:v3

If the docker container is launched successfully, the isolated sinergym environment is enabled. Navigate to the to the source directory dc-rl to be able to execute DCRL-Green:

cd dc-rl

Note

Some useful Docker commands could be found here

Next, run the script:

EPLUS=1 python train_ppo.py

Note

The episode_reward_mean will display nan values for the first few iterations until 1 episode is completed

2.3 Running in background mode

If you want to run the DCRL-Green framework in background mode use the following command:

nohup python PYTHON_SCRIPT > OUTPUT_FILE.txt  &

where PYTHON_SCRIPT is the script you want to run (e.g., train_ppo.py) and OUTPUT_FILE is the name of the file that will contain the output (e.g. latest_experiment_output).

3. Monitor the results

The training logs and the results of each trial are stored in the specified local directory, under a sub-folder called results and can be visualized with TensorBoard by specifying the output directory of the results.

Example:

tensorboard --logdir ./results

In this example, test is the default name of the experiment.
