============
Installation
============

DCRL-Green enables data center simulation framework hosted using OpenAI Gym. The framework is self-contained but allows for the integration of other co-simulation models such as EnergyPlus :cite:p:`crawley2000energy` through a plug-in interface.
DCRL-Green's custom simulation model provides users with advanced customization options for DC designs, including the ability to specify individual servers within racks. More details on the framework configuration is provided in the :ref:`Environment`
section.
