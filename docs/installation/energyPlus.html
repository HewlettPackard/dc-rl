EnergyPlus enabled DC simulation

Dependencies

  1. Python

    Python version 3.x is recommended.

  1. Docker

    We include a Dockerfile for installing and setting up the image for running Sinergym [4].

  1. TensorBoard

    DCRL-Green uses Ray RLlib to execute reinforcement learning algorithms. By default, RLlib logs results in Tensorboard. RLlib does not automatically install a deep-learning framework, but supports TensorFlow (both 1.x with static-graph and 2.x with eager mode) as well as PyTorch. Depending on your needs, make sure to install either TensorFlow or PyTorch (or both).

First-time setup

Clone the latest DCRL-Green version from github using:

git clone https://github.com/HewlettPackard/dc-rl.git

Pull the Docker image with command:

sudo docker pull agnprz/carbon_sustain:v3

A docker container can be launched using the command:

sudo docker run -t -i -v ./:/sinergym/dc-rl --shm-size=10.24gb agnprz/carbon_sustain:v3

If the docker container is launched successfully, the sinergym environment is enabled. A python script can be executed by navigating to the directory dc-rl.
