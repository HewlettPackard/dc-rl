The following table describes the action space of the three agents: Load shifting agent (agent_ls), DC cooling agent (agent_dc) and the Battery controlling agent (agent_bat).

Source

Size

Type

Range

Description

agent_ls

2

Discrete

[0,1]
0 - Assign load at the current time step from the flexible load budget
1 - Do not shift flexible load (Idle)

agent_dc

9

Discrete

[0,8]
0 - Reduce temperature by 5 units
1 - Reduce temperature by 2 units
2 - Reduce temperature by 1 unit
3 - Reduce temperature by 0.5 unit
4 - No change in temperature (Idle)
5 - Increase temperature by 0.5 unit
6 - Increase temperature by 1 unit
7 - Increase temperature by 2 units
8 - Increase temperature by 5 units

agent_bat

3

Discrete

[0,2]
0 - Use the charge function to store energy in the battery
1 - Use the discharge function to release energy from the battery
2 - Do nothing (Idle)
